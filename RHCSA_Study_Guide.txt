Understand and use essential tools
------------------------------------
complete | Access a shell prompt and issue commands with correct syntax | Solution: Through GUI its in top left Applications > Utilities > Terminal
complete | Use input-output redirection (>, >>, |, 2>, etc.) | Solution: > sends output to file, >> appends output to file, | pipes output to a command, 2> outputs standard error to a file.
complete | Use grep and regular expressions to analyze text | Solution: man grep
complete | Access remote systems using ssh | Solution: man ssh
complete | Log in and switch users in multiuser targets | Solution: ssh user@host; su user2switch
complete | Archive, compress, unpack, and uncompress files using tar, star, gzip, and bzip2 | Solution: Create gzip: tar -czvf nameoftar.tar /directories /to /tar, Create bzip2: tar -cjvf nameoftar.tar /directories /to /tar
complete | Create and edit text files | Solution: Vi is a text editor installed on every linux system
complete | Create, delete, copy, and move files and directories | Solution: Delete files: rm /file/to/remove, Move files: mv /file/to/move /new/location, Copy files: cp /file/to/copy /new/copy
complete | Create hard and soft links | Solution: ln -s /target/to/create/link link_name, -s is for soft links
complete | List, set, and change standard ugo/rwx permissions | Solution: man chmod
complete | Locate, read, and use system documentation including man, info, and files in /usr/share/doc | Solution: use man for quick documentation. Other import files are generaly found in /usr/share/doc
Note: Red Hat may use applications during the exam that are not included in Red Hat Enterprise Linux for the purpose of evaluating candidate's abilities to meet this objective.
